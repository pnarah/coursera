# 02 Mobile Project Bootstrap

## Objective
Initialize Flutter (recommended) mobile application with base dependencies and navigation.

## Prerequisites
- Flutter SDK installed
- Project structure from Task 01

## Deliverables
- Created Flutter app in `mobile/`
- Added dependencies: riverpod, dio, go_router, flutter_secure_storage
- Basic folder structure (`lib/core`, `lib/features/authentication`)

## Suggested Steps
1. Run `flutter create mobile`.
2. Add dependencies in pubspec.yaml.
3. Implement `main.dart` with Riverpod and go_router baseline.
4. Add placeholder LoginScreen and HomeScreen.

## Prompts You Can Use
- "Scaffold Flutter app with Riverpod and go_router."
- "Add secure storage service to Flutter app."
- "Create initial navigation routes for login and home."

## Acceptance Criteria
- App builds successfully.
- Can navigate between login and home placeholder screens.
